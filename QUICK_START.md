# Quick Start

## Installation (One Command)
```bash
./install.sh
```

## Usage

### Activate Environment
```bash
source .venv/bin/activate
```

### Run Training

**Recommended (module syntax):**
```bash
python -m alpha_zero.main           # Basic training
python -m alpha_zero.run_training   # With auto-restart
```

**Alternative (direct):**
```bash
python alpha_zero/main.py           # Basic training
python alpha_zero/run_training.py   # With auto-restart
```

### Deactivate When Done
```bash
deactivate
```
